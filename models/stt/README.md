# STT Models Directory
Place your Sherpa-ONNX speech-to-text models here.

## Recommended Models

### Zipformer Streaming (for NPU)
Download from: https://github.com/k2-fsa/sherpa-onnx/releases
Files needed:
- `zipformer-encoder.onnx`
- `zipformer-decoder.onnx`
- `zipformer-joiner.onnx`
- `tokens.txt`

### Whisper (for CPU/CUDA)
Download from: https://github.com/k2-fsa/sherpa-onnx/releases
Files needed:
- `whisper-base-int8-encoder.onnx` (CPU)
- `whisper-base-int8-decoder.onnx` (CPU)
- `whisper-small-encoder.onnx` (CUDA)
- `whisper-small-decoder.onnx` (CUDA)
- `tokens.txt`
